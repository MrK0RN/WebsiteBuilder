Цель: Разработать веб-сайт для подбора пластика по заданным характеристикам с возможностью перехода на сайт продавца.  

#### Основные страницы и функционал:  

1. Страница регистрации  
   - Поля:  
     - Имя пользователя  
     - Электронная почта  
     - Пароль (с подтверждением)  
     - Опционально: компания, должность  
   - Валидация данных (корректность email, сложность пароля)  
   - Кнопка "Зарегистрироваться"  
   - Ссылка на страницу авторизации ("Уже есть аккаунт? Войти")  

2. Страница авторизации  
   - Поля:  
     - Электронная почта / Логин  
     - Пароль  
   - Кнопка "Войти"  
   - Ссылка на восстановление пароля  
   - Ссылка на регистрацию ("Нет аккаунта? Зарегистрироваться")  

3. Главная страница (поиск пластика)  
   - Фильтры для подбора:  
     - Тип пластика (ABS, PLA, PETG, нейлон, поликарбонат и др.)  
     - Основные характеристики:  
       - ПТР (Показатель текучести расплава, г/10 мин)  
       - Температура плавления (°C)  
       - Плотность (г/см³)  
       - Прочность на разрыв (МПа)  
       - Ударная вязкость (кДж/м²)  
       - Термостойкость (по Вика или другим стандартам)  
       - Цветовая гамма (выбор из палитры или текстовый фильтр)  
       - Наличие сертификатов (FDA, UL94 и др.)  
     - Производитель (фильтр по брендам)  
   - Кнопка "Найти"  
   - Блок с популярными/рекомендованными пластиками  

4. Карточки пластиков (вывод в виде списка/сетки)  
   - Краткая информация:  
     - Название пластика  
     - Производитель  
     - Основные характеристики (ПТР, температура плавления, прочность)  
     - Миниатюра (изображение образца)  
   - Кнопка "Подробнее" (переход на страницу пластика)  
   - Кнопка "Купить" (редирект на сайт продавца)  

5. Страница подробного описания пластика  
   - Полное название и производитель  
   - Фото/3D-модель материала  
   - Все технические характеристики:  
     - Механические (прочность, удлинение при разрыве, модуль упругости)  
     - Термические (температура плавления, термостойкость)  
     - Химические (устойчивость к растворителям, влагопоглощение)  
     - Электрические (если актуально)  
     - Эксплуатационные (применение, совместимость с пищевыми продуктами и др.)  
   - Ссылки на официальную документацию (PDF-паспорта)  
   - Список продавцов с ценами и кнопками перехода  
   - Возможность добавить в избранное (для авторизованных пользователей)  

6. Панель администратора / Добавление нового пластика  
   - Доступ только для авторизованных модераторов/админов  
   - Форма добавления:  
     - Название, производитель  
     - Загрузка изображений  
     - Ввод всех характеристик (ПТР, плотность, прочность и т. д.)  
     - Ссылки на продавцов (URL + цена)  
     - Загрузка технической документации  
   - Кнопка "Опубликовать"  

7. Дополнительный функционал:  
   - Избранное (сохранение пластиков в личный кабинет)  
   - Сравнение нескольких пластиков (таблица с характеристиками)  
   - Комментарии/отзывы пользователей (если нужно)
